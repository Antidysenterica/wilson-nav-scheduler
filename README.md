# WILSON
WILSON: Web Interface for Location, Scheduling, and Office Navigation

A Campus Map + Appointment Scheduler for Ateneo de Naga University 

This system is designed to help students, faculty, staff, and visitors easily navigate university facilities and efficiently schedule appointments with campus offices and personnel. 

Website Structure:
**WILSON Website**
│ 
├── Landing Page (View Campus Map/Login)
│       ├── Login
│       ├── Create Account
│       │       ├── Create Account Page
├────── ┘ 
│ 
├── Campus Map Tab
│      └── Floor
│               ├── Room Details Option
│               ├── Appoint Schedule Option (For all, except non-account users)
│               │      └── Available Time Slot Tab (Available Slots)
│               │               └── Confirm Time Slot
│               │ 
│               └── Manage Appointments Option (Faculty/Staff only)
│                        ├── Approve Pending Requests
│                        └── Edit Available Time Slots
│ 
└── User Profile
         ├── Edit Profile
         ├── Change Password
         ├── About
         └── Log Out
